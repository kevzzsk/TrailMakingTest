# Trail Making Test
TMT is a web app used to diagnose patients with dementia. Patients will go through several round of TMT.

# Demo
Visit https://kevzzsk.github.io/TrailMakingTest/

![Website](https://img.shields.io/website?down_color=red&down_message=OFFLINE&style=for-the-badge&up_color=green&up_message=ONLINE&url=https%3A%2F%2Fkevzzsk.github.io%2FTrailMakingTest%2F)


# Local Environment Setup
```bash
# To start localhost server
npm start

# To build and deploy to gh-pages
npm run update

# To generate docs
npm run createdoc
```
# Docs
Docs is generated by JSDoc on ./Docs
